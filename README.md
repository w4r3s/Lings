# Lings

## Local dev

Steps to set up the local dev environments:

1. Open the current folder in a dev container
2. `poetry install`
3. `poetry run python lings/gui.py`

Or you can run it the `just` way:

1. `just install`
2. `just run`

Run `just help` for more usages.
